# Backup recording — 2-minute screen capture

If Wi-Fi, camera, or the venue projector misbehaves, play this video
instead of running the live demo.

## What to capture

Follow `DEMO_SCRIPT.md` but condensed:

1. (0:00–0:20) Open the dashboard. Show the three running terminals.
2. (0:20–0:50) Golden path — Karim pays on bKash. Show PAID + green SMS.
3. (0:50–1:20) Enforcement — hold Rahim's card, click "Run enforcement
   tick" four times, show the red BRTA-block row land.
4. (1:20–1:50) Tab through Events / Vehicles / Analytics.
5. (1:50–2:00) End on the Analytics revenue-vs-outstanding chart.

Keep narration short and factual — the handout has the pitch.

## How to record on Windows

### Option A — Xbox Game Bar (built-in, zero install)

1. Press `Win + G` → accept "Yes, this is a game" if it asks.
2. Capture widget → click the record button (or `Win + Alt + R`).
3. Stop with the same shortcut. MP4 lands in
   `C:\Users\<you>\Videos\Captures\`.

Limitation: captures one window only. Maximise the dashboard first.

### Option B — OBS Studio (recommended, free)

1. Install from https://obsproject.com
2. Scene → Add → **Display Capture** (captures the whole screen,
   including the phone's IP-Webcam view).
3. Settings → Output → Recording → set format to `mp4`, quality
   "Indistinguishable Quality".
4. Start Recording → run the demo → Stop Recording.

### After recording

- Trim with the built-in **Photos** app (Windows) — open the file, click
  "Edit & Create" → "Trim".
- Export at 1080p, drop into the same `demo/` folder as
  `demo/shohojtoll_backup.mp4`.

## Playback plan
